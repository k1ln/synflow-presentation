# Synflow presentation backbone

This folder contains a first Reveal.js backbone for:

**Modular Synth in the browser.**

## Files
- `index.html` — Reveal deck scaffold with title + 18-minute overview
- `theme.css` — projector-friendly custom styling (high contrast, large typography)

## Run
Open `index.html` in a browser.

If you prefer a local server (recommended), run from this folder:

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.
